To-Do List Web Application

This project is a simple, browser-based **To-Do List** application developed using HTML, CSS, and JavaScript.

### Features

* **Add tasks**: Users can enter and add new tasks to the list.
* **Remove tasks**: Each task includes a button that allows users to delete it from the list.
* **Search functionality**: Includes a real-time search bar that filters tasks based on the user's input.
* **Interactive user interface**:

  * Styled components and responsive layout.
  * Visual feedback with hover effects on buttons.
* **Dynamic task rendering**: Uses JavaScript to dynamically update the task list without refreshing the page.

### Technologies Used

* **HTML** – for structuring the application layout
* **CSS** – for styling the interface and enhancing the user experience
* **JavaScript** – for functionality such as task management, filtering, and DOM manipulation
